## [0.1.0] - 2026-04-19

### Added

- Initial release of sokol-ts: a Sokol-inspired WebGPU rendering and app framework for TypeScript.
- `run` — application entry point that initializes and drives the render loop.
- `createGfx` — creates a WebGPU graphics context.
- Full TypeScript type definitions and declaration maps shipped with the package.
- ESM-only output targeting ES2022.
- npm provenance support via GitHub Actions publish workflow.
